# ginoitaliano.github.io

Personal landing page : live at [ginoitaliano.github.io](https://ginoitaliano.github.io)

Built with vanilla HTML/CSS. No frameworks, no dependencies, no build step. 

## Stack

- HTML5 + CSS3 (custom properties, keyframe animations, grid/flex)
- [Bebas Neue](https://fonts.google.com/specimen/Bebas+Neue) + [DM Sans](https://fonts.google.com/specimen/DM+Sans) + [DM Mono](https://fonts.google.com/specimen/DM+Mono) via Google Fonts
- Zero JavaScript, zero dependencies
